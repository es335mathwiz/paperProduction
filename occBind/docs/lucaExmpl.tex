For example, consider the simple model


\begin{gather*}
q_{t} +\beta_p(1 - \rho_p)q_{t + 1} + \rho_pq_{t - 1} - \sigma_pr_{t} +
     r_{ut}=0\\
 r_{t} = \max (\bar{r}, \phi_pq_{t}) \\
 (r_{ut}-r^\ast) = \rho_{ru} (r_{ut - 1}-r^\ast) + \eta \epsilon_{y,t}
\end{gather*}

Then 


\begin{gather*}
  x_t=\xtVec
\end{gather*}
